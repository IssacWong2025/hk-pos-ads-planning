三、Meta 廣告形式管理層速覽

總覽：Meta 生態的主要承載版位是 Facebook 與 Instagram 的 Feed、Reels、Stories，以及 Messenger / WhatsApp 消息入口。首月目標類型以「線索（名單）與消息對話」為主，觸達、流量、互動作為補充，不以品牌曝光作為首要成敗判斷。

形式 / 版位組合 適合目的 優點 風險 / 限制
Feed 圖片 / 短影片（FB+IG） 觸達、流量、初步諮詢 覆蓋穩定、素材製作成本可控 點擊後流失較高，需靠後續承接
Reels（IG/FB） 拓展新客、擴量 觸達效率高，適合冷受眾 受素材品質影響大，波動明顯
Stories（IG/FB） 快速導流、短期促動 互動直覺，時效性強 訊息停留短，需明確行動指令
Lead Ads（即時表單） 線索獲取（名單） 填單阻力低，易控 CPL（每條線索成本） 名單品質可能參差，需有效諮詢篩選
Click-to-Message（WhatsApp / Messenger） 消息獲客、即時對話 可直接進入對話，便於 5 分鐘首響 需銷售在線，否則回覆延遲會掉單

首月主選與備用方案

  • 主選組合：Lead Ads（主）+ Click-to-Message（輔）。Lead Ads 保障名單量，消息廣告承接高意向客，與「5 分鐘首響、24h 3 次跟進」SLA 匹配。
  • 投放理由：首月是可控試驗，需同時驗證「量」與「承接效率」，組合方案比單一路徑更穩定。
  • 備用切換：若在 HKD 2,000 節點出現 CPL 高於 HKD 160 或有效諮詢率低於 35%，將 20%-30% 預算切至 Feed/Reels 影片導流測試，補充受眾與素材驗證。

五、風險與歸因可視化

P1/P2 風險優先級

  • P1：諮詢量不足（直接影響闸門推進與樣本有效性）
  • P2：有效諮詢占比偏低（名單多但可成交線索不足）

未達標歸因流程

  • 步驟 1：流量（諮詢量） -> 是否先天量不足
  • 步驟 2：質量（有效諮詢率） -> 是否流量結構失衡
  • 步驟 3：承接激活（SLA 達成） -> 是否後鏈路承接問題
