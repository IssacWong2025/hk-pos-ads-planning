本版供管理層快速判斷：這輪投放是否形成可複製模式，並能否為後續標杆案例建立穩定來源。

經營決策視角

一、首月目標與商業意義

硬結果目標
  • 首月有效激活：5 位
  • 不拆來源，直接看總結果
廣告端通過線
  • 首月諮詢量：40-60 條（目標值 50）
  • 預算：HKD 6,000（Meta only）
主商業價值
  • 主：建立標杆客戶基礎（可對外對內複用）
  • 次：驗證可複製獲客模型（為下月放大做準備）
首月原則
  • 先驗證方向與閉環，不設定固定 ROI 硬線
  • 決策依據：諮詢節點達成與激活進度

二、策略主線（先流量，後結果）

層級本期策略管理層判斷問題
流量層 先把諮詢量穩定到 40-60 區間。 是否跑出足夠需求盤？
質量層 在達量前提下提升有效諮詢占比。 是否避免低質流量消耗？
結果層 推進有效激活累計至 5 位。 是否形成可複製的轉化閉環？

三、分階段動作與里程碑

階段核心動作階段產出推進條件
冷啟動期 快速建立詢盤來源，辨識可持續起量方向。 諮詢量開始穩定產生 達到節點諮詢門檻
驗證期 保留有效組合，壓縮低效消耗，提升有效諮詢比例。 有效諮詢與激活效率同步改善 有效激活連續出現
放大期 擴大已驗證組合，控制風險下衝刺結果。 有效激活累計逼近或達到 5 位 節點結果持續達標

四、節點機制（預算主軸 + 時間兜底）

節點類型用途決策動作
預算節點（主） 按消耗進度檢查諮詢達成狀態。 達標：按計畫推進；未達：立即復盤與調整。
時間節點（兜底） 避免未達消耗節點時決策延後。 固定頻率檢查方向，防止節奏失控。
停損信號（已確認）
  • 當消耗達到節點，但諮詢總量低於該節點門檻時，立即啟動策略復盤。
  • 復盤只看三件事：受眾方向、預算節奏、階段目標。

五、主要風險與預案

風險優先級風險描述預案方向
P1 諮詢量不達標（主風險） 優先調整投放方向與預算節奏，先恢復詢盤能力。
P2 諮詢有量但質量偏弱（次風險） 在不犧牲總量前提下，收斂低質來源並提升有效占比。

六、追加預算機制

  • 首月基礎預算維持 HKD 6,000。
  • 若節點結果連續達標，可提出追加預算申請。
  • 追加前提：方向成立（諮詢穩定）且結果在向 5 位激活目標收斂。

七、未達標時的歸因順序（管理層統一口徑）

順序先檢查項判斷目的
1流量（諮詢量）確認是否先天量不足
2質量（有效諮詢占比）確認是否流量結構失衡
3承接激活（諮詢後轉化）確認是否後鏈路承接問題
